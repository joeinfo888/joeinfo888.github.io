Using Rsync instead of SCP to transfer files between two servers

22 Nov 2012 on miscellaneous

SCP is a great tool for transfering files using ssh protocol. I had been using it for transfering files from one server to another for a long time. But recently I had been working on a few servers, with very unstable connectivity. The connection had been breaking half way through the transfer. One of the problems with SCP is that you cannot resume a file transfer that got interrupted halfway. So I use rsync for this purpose. Following is an expample

rsync --partial --progress --rsh='ssh -p 1234' joe@example.com:/home/joe/latest.zip .
