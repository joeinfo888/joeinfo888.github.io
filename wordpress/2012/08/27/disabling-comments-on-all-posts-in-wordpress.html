Disabling comments on all posts in wordpress

27 Aug 2012 on wordpress

A few days before I started managing a wordpress blog with thousands of posts. The blog had been ignored for a few months and was full of spammy comments. The client asked me to disable comments on all posts for a few weeks. It is practically impossible to disable comments on thousands of posts by taking each individual post and unchecking allow comments checkbox. I figured my way around by executing an SQL query in the blog database to disable comments on all posts. Following are the steps to disable comments on all posts of a wordpress blog at once.

  1. Access the mysql database used by the blog using phpmyadmin or a similar tool
  2. Execute the following sql query
UPDATE wp_posts
SET comment_status = 'closed', ping_status = 'closed'
WHERE comment_status = 'open';

Please note that wp_ in wp_posts is the table prefix used in the database and it may vary according to the table prefix you chose during wordpress installation.
